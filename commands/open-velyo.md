Open Velyo (PM Workspace) in the browser. First check if the server is running on port 3000. If it's not running, start it by running `node server.js` from `pm-workspace/action-tracker/` in the background, wait a moment, then open the browser. If it's already running, just open the browser directly.

Run this command to open Velyo:
```
open http://localhost:3000
```

After opening, confirm to the user that Velyo is now open in their browser.
